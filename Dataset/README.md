# Dataset

This folder contains the datasets used in the Walmart Sales Data Analytics project.

## Files

### Walmart.csv
- Original raw dataset used for analysis.

### walmart_clean_data.csv
- Cleaned dataset after preprocessing and feature engineering.

## Dataset Features

- Invoice ID
- Branch
- City
- Category
- Unit Price
- Quantity
- Date
- Time
- Payment Method
- Rating
- Profit Margin
- Total Sales

## Dataset Source

The dataset is based on Walmart sales transaction records and is used for educational and data analytics purposes.
