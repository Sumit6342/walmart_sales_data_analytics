# SQL Queries

This folder contains all the SQL queries used in the Walmart Sales Data Analytics project.

## Database
- MySQL

## File

- `walmart_sales_queries.sql` – Contains all SQL queries used for business analysis.

## Business Questions Solved

1. Find different payment methods and number of transactions.
2. Find the highest-rated category in each branch.
3. Identify the busiest day for each branch.
4. Calculate total quantity sold by payment method.
5. Determine average, minimum and maximum ratings for each city.
6. Calculate total revenue by month.
7. Find the month with the highest Cost of Goods Sold (COGS).
8. Identify the city with the highest revenue.
9. Determine the product category with the highest profit margin.
10. Analyze sales performance by branch and category.
11. Find customer purchasing patterns.
12. Compare revenue across different branches.
13. Identify top-performing categories based on sales.
14. Analyze rating trends across branches.
15. Generate business insights using SQL aggregations and window functions.

## SQL Concepts Used

- SELECT
- WHERE
- GROUP BY

- CASE WHEN
- Aggregate Functions
