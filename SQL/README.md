# SQL Queries

This folder contains the SQL scripts used for data preparation, feature engineering, and business analysis in the Walmart Sales Data Analytics project.

## Database
- MySQL

## SQL Operations Performed

- Database creation
- Data import
- Data cleaning
- Feature engineering
- Exploratory Data Analysis (EDA)
- Business problem solving
- Sales analysis
- Customer analysis
- Product analysis

## File

- `MYSQL Queries.sql` – Contains all SQL queries used in this project.
